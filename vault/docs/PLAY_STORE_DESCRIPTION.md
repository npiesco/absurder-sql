# VaultApp Play Store Listing Draft

VaultApp

## Short Description (80 chars max)
Offline password manager with encrypted vault, TOTP codes, and biometric unlock.

## Full Description
VaultApp is a local-first password manager built for speed, privacy, and reliability.

Your vault stays on your device, encrypted at rest, with tools for secure credential management and fast daily access.

Key features:
- Encrypted local vault for passwords and sensitive notes
- Master password unlock with biometric support (fingerprint)
- Built-in TOTP authenticator support
- Password generator with configurable strength
- Folders, favorites, tags, sorting, and search
- Import/export backup workflow for manual sync and recovery
- Security-focused UX: auto-lock, clipboard clear options, and audit indicators
- Works offline after install

VaultApp is designed for users who want strong security without cloud lock-in.
